# WB AI Prompts – On-the-Go Whitepaper Writing

These are ready-to-use prompt templates based on Whitepaper Blueprint v1.1.

**How to use (anywhere – mobile/desktop):**

1. Open Grok/Claude/ChatGPT/etc. (even on phone)
2. Upload the Whitepaper-Blueprint PDF (from this repo) or paste its text
3. Copy one of these prompt files
4. Paste into AI + fill in [YOUR TOPIC] / [YOUR IDEA]
5. AI writes the full whitepaper in Markdown
6. Save draft → push to whitepapers/ folder in your fork → get NOMOS + CRP safety reports
7. Publish to Medium / AION repo

**Prompts included:**
- 01-Basic-Prompt.md → Quick general whitepaper
- 02-Deep-Dive-Prompt.md → Technical/research focused
- 03-Safety-Enhanced-Prompt.md → For high-stakes AI safety topics

All prompts enforce blueprint structure + evidence rigor + humility.
